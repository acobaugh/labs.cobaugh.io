## Specs

* Input: 240VAC, 50A, single phase
* Output: 680VDC, ~15A
* Configuration 
  * Voltage doubler
  * Semikron SKKD 81/16 dual diode
  * 2x 4700uF, 450VDC
* Features
  * Automatic, time-based resistor pre-charge
  * Automatic resistor discharge
  * 240VAC auxillary output
  * Remote emergency stop
